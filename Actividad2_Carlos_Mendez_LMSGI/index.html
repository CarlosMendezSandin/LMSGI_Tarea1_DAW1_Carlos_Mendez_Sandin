Descripción de la página

Esta página consiste en unos 8 ejercicios, como se puede ver en el navegador, en los cuales aprenderemos a utilizar etiquetas o marcas de HTML con una semantica especifica y adecuada

Breve descripción de los ejercicios

Ejercicio 1

En este primer ejercicio voy a tratar los distintos tipos de encabezados (o títulos) que existen en HTL 5 para una página web.

Ejercicio 2

Empiezo a utilizar distintas marcas HTLM 5 para modificar o resaltar ciertas palabras de texto. Como la marca "pre" o "code"

Ejercicio 3

Sigo utilizando distintas marcas HTLM 5 para modificar o resaltar ciertas palabras de texto. Como "ins", "del" y otras

Ejercicio 4

En este ejercicio conociendo ya ciertas etiquetas de marcas de HTML 5 confecciono un mini curriculum vitae

Ejercicio 5

En este ejercicio comienzo con la introducción de imágenes con las marcas “figure”, "figcaption" y otras

Ejercicio 6

Sigo con marcas HTML para confeccionar una pagina web

Ejercicio 7

Confecciono un mini curriculum vitae más completo

Ejercicio 8

Confección de una pagina web
